# Expense Tracker

A personal finance tracking web app built with 
React, TypeScript and Tailwind CSS.

## Live Demo
🔗 [expense-tracker.vercel.app](https://expense-tracker-phi-inky.vercel.app/)

## Features
- Add income and expense transactions
- Real time balance, income and expense totals
- Spending breakdown chart by category
- Filter by category and sort transactions
- Data persists with localStorage
- Clear all transactions with confirmation
- 🌐 Deployed on Vercel — CI/CD via GitHub

## Tech Stack
- React + TypeScript
- Tailwind CSS
- Recharts (data visualization)
- useContext (state management)
- localStorage (data persistence)
- Vite + Vercel

## Getting Started
```bash
git clone https://github.com/RichieErnie/expense-tracker
cd expense-tracker
npm install
npm run dev
```
